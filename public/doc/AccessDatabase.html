Overview

This class is used to read to and write from Access Databases (both mdb and accdb)

Examples:

# connect to database
database_file   = 'D:/temp/my_database.accdb'
access_database = AccessDatabase.connect(database_file)

# check that my_table exists
Logger.raise("table 'my_table' doesn't exist") unless access_database.table_exists?('my_table')

# read data into an array (where each record is itself an array)
some_data       = access_database.execute("SELECT * FROM [my_table]")

# read data into an array of hashes, where the field names (as symbols) are used as keys
opts = { :keys_as_symbols => true, :as_hash => true }
some_data = access_database.execute("SELECT * FROM [my_table]", opts)
