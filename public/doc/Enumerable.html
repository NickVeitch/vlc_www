- (Object) as_index_for(&value_function)

Takes something enumerable and uses it as the key into a hash where the value comes from evaluating the given block This is very useful when you have something like this:

Examples:

[1,2,3].map { |i| [i, i * 17 * 42] }.to_hash   # => {1=>59, 2=>76, 3=>93}
# which can be written more concisely as
[1,2,3].as_index_for { |i| i * 17 + 42 }       # => {1=>59, 2=>76, 3=>93}

def as_index_for(&value_function)
  inject({}) do |accum, elem|
    accum[elem] = value_function.call(elem)
    accum
  end
end

- (Object) group_by_indices(indices = [0])

This is a funky method - for each element (which must be an array), it will construct a key by extracting the specified indices from that array and storing the remainder in a hash from key -> [remainder1, remainder2, etc]

Examples:

[[0,1], [0,2], [1,2]].group_by_indices          # => {0 => [1,2], 1 => [2]   }
[[0,1], [0,2], [1,2]].group_by_indices([1])     # => {1 => [0], 2 => [0,1]   }
[[0,1,2,3], [0,1,3,4]].group_by_indices([0,1])  # => {[0,1] => [[2,3],[3,4]] }

def group_by_indices(indices = [0])
  return Hash.new_with_default([]) if size < 1
  data_rows = (0..(first.size-1)).to_a - indices
  inject(Hash.new_with_default([])) do |h,row|
    key = indices.map { |i| row[i] }
    key = key.size == 1 ? key.first : key
    data = data_rows.map { |i| row[i] }
    h[key] << (data.size == 1 ? data.first : data)
    h
  end
end

- (Object) split

Split an array in to sub-arrays based on the occurance of a specified value. i.e. an array implementation of the String split(delimiter) method. Implementation taken from http://stackoverflow.com/questions/4800337/split-array-into-sub-arrays-based-on-value

Examples:

(1..9 ).to_a.split{ |i| i%3==0 } #=> [[1, 2], [4, 5], [7, 8]]
(1..10).to_a.split{ |i| i%3==0 } #=> [[1, 2], [4, 5], [7, 8], [10]]

def split
  result = inject([[]]) do |a,v|
    yield(v) ? (a << []) : (a.last << v)
    a
  end
  result.pop if result.last.empty?
  result
end
