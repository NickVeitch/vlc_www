Overview

The ZenithModelRun class is used to execute all Zenith Model Runs! It contains numerous properties which can control all aspects of the model run process, input parameters, and outputs.

Examples:

zenith_model_run = ZenithModelRun.new
zenith_model_run.do_assignment = true
zenith_model_run.execute
